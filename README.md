# myApp
myApp is a free test app made with `electron` in `html`/`css`/`js`.
## Acknowledgements
- [README.md](https://github.com/moony404/myApp/blob/master/README.md)
## Installation
Run myApp with npm command line :
```bash
npm start
```
Build myApp with npm command line :
```bash
npm run package-win
```
